   </div>  </form></body></html>``````cssinput[type="url"] {  z-index: 3;}.invalid-message {  background: rgba(255, 0, 0, .2);  left: 0;  opacity: 0;  padding: 8px;  pointer-events: none;  position: fixed;  top: 0;  transition: opacity .2s;  width: 100%;  z-index: 0;}  :invalid ~ .invalid-message {  opacity: 1;  pointer-events: auto;  font-size: 2em;  color: red;  z-index: 1;}```注意这里用到的`:invalid`伪类，以及`~`选择符（表示子元素）。`pattern`是HTML5规范中类型为`email`的`input`的属性，用于验证输入符合的规范。`aria-describedby`属性用于输入框的可用性描述，对应ID为`#message`的DIV元素，这个元素的ARIA属性为`role="tooltip"`。首先，默认隐藏信息提示的内容，接着，为了加上渐现的效果，使用了CSS的不透明度过渡。###  <input type="email" class="input-text" id="name2" placeholder="Your Email" />  <label class="label-helper" for="name2">Your Email</label></div>```如我们所见，HTML没有什么花样。唯一不同寻常的是`<label>`标签出现在`<input>`后面。接着是相应的CSS：```css.form-row {  position: relative;}.input-text {  background-color: #FFF;  border: 1px solid #CCC;  margin-bottom: 8px;  padding: 8px 4px;  position: relative;  width: 100%;  z-index: 3;}.label-helper {  bottom: 0;  left: 0;  opacity: 0;  position: absolute;  transition: .2s bottom, .2s opacity;  z-index: 1;}```CSS的第一部分是`form`元素的默认样式，`.form-row`样式让输入框附近的`.label-helper`保持绝对定位。`.input-text`元素为相对定位而且应用了`z-index`值，因为我们不想让`label`在输入框获取了焦点时变换到输入框前面。```css.input-text:focus + .label-helper,.input-text:invalid + .label-helper {  bottom: 95%; /* magic number, boo */  font-family: arial;  font-size: 14px;  line-height: 1;  opacity: 1;  padding: 4px;}.input-text:invalid {  border-left: 10px solid #F00;}.input-text:invalid + .label-helper:after {  color: #F00;  content: 'X';  font-family: arial;  font-size: 14px;  line-height: 1;  padding-left: 12px;}```这里最重要的代码片段就是`:focus`的使用以及相邻兄弟选择器`+`的使用（兄弟选择器是我们大部分demo用到的内容）。基本上，在输入框获取了焦点是，兄弟节点（label）的透明度会变换而且会被重新定位到input上方，给没有placeholder标签的输入框提供上下文。额外的一点是，还提供了email输入框的无效状态。因为输入框不支持伪类元素，因此在`label`上应用了错误“x”。## type="radio" class="radio" name="pages" id="exit" checked><div class="page demo2">  <input id="page_1" class="radio" type="radio" name="pages">  <section class="section-container section-one">    <label class="check-label exit-label" for="exit">X</label>    <label class="page-label check-label" for="page_1">      <span>Page X</span>    </label>    <header class="section-header">      <div class="section-content">        <h1>Title of Section X</h1>      </div>    </header>    <div class="section-info">      <div class="section-content">        <p></p>      </div>    </div>  </section>  <!-- repeat this 3 more times, with IDs of "page_2, page_3, and page_4" --></div>```我们注意到在每节前面用到了`<input type="radio">`元素。我们将在CSS里大量使用这些元素，因为我们需要再次依赖相邻兄弟选择器（`+`）来让变换工作。基本上我们会使用`radio`按钮的`:checked`状态，通过点击相关联的`label`以及兄弟选择器来激活，以只通过CSS来修改结构。CSS如下：```css.radio,.exit-label {  display: none;  height: 0;  visibility: hidden;}.exit-label {  border: 1px solid #333;  color: #444;  display: inline-block;  padding: 4px 12px;  text-decoration: none;}.page-label {  height: 100%;  left: 0;  position: absolute;  text-align: center;  top: 0;  width: 100%;}.page-label span {  position: relative;  top: 50%;}```上面的CSS会处理按钮的样式并且隐藏radio按钮。`.page-label`和`.page-label span`元素用于大面板上的按钮，虽然`.exit-label`类为关闭按钮定义了样式。### 原文<http://www.sitepoint.com/you-can-do-that-with-css/>
